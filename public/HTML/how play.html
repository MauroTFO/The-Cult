COMO JOGAR

Utilize as teclas W,A,S,D para movimentação do personagem:

  • W -> Movimentar para cima
  • A -> Movimentar para esquerda
  • S -> Movimentar para baixo
  • D -> Movimentar para direita

Utilize as teclas E e G para interação com o ambiente:

  • E -> Para pegar um item
  • G -> Para interagir
